## Возможности (и как они работают)

- **Вход по QR-коду** — Отсканируйте QR-код приложением Discord на телефоне, чтобы войти мгновенно. На телефоне: *Discord → Настройки → Сканировать QR-код*, затем наведите на код в панели. Без ввода пароля на Deck.
- **Полноэкранный вход (запасной)** — Открывает Discord на весь экран для входа по почте/паролю или решения CAPTCHA, когда QR недоступен.
- **Голосовой чат** — Заходите в голосовые каналы и слышьте всех; каждый участник показан в реальном времени (кольцо речи, значки заглушения/оглушения) и ползунок громкости на каждого (0–200 %).
- **Личные сообщения (ЛС и групповые)** — Просматривайте беседы и начинайте/присоединяйтесь к голосовым звонкам с друзьями прямо из меню быстрого доступа. Активные звонки выделяются.
- **Голосовой обозреватель серверов** — Смотрите, в каких голосовых каналах есть люди (с аватарами), прежде чем зайти.
- **Заглушить / Оглушить / Отключиться** — Голосовое управление одним касанием из QAM.
- **Go Live (демонстрация экрана)** — Делитесь всем экраном в голосовом канале.
- **Ретрансляция микрофона** — Ваш микрофон захватывается в интерфейсе Steam и передаётся в Discord, так что вас слышат, хотя Discord работает в скрытой фоновой вкладке. Вход и выход автоматически следуют за вашим устройством по умолчанию (подключите гарнитуру — переключится само).
- **Игровой статус** — Показывает игру, в которую вы играете, как ваш статус в Discord.
- **Уведомления в игре** — ЛС и упоминания появляются как уведомления Steam.
- **Push-to-talk** — С физической кнопкой (по умолчанию R5).
- **Отправка скриншотов** — Отправьте скриншот Steam в любой канал Discord.
- **[Vencord](https://vencord.dev/)** внедряется автоматически, давая доступ к его экосистеме плагинов.

## Как работает звук (самое сложное)

Discord работает в **скрытом** представлении браузера внутри Steam. Две вещи заставляют голос работать:

1. **Слышать других** — Chromium приостанавливает звук в скрытых вкладках (политика автовоспроизведения). Streamcord возобновляет звук Discord с помощью симулированного через CDP жеста пользователя, чтобы входящий голос действительно воспроизводился на вашем выходе по умолчанию.
2. **Быть услышанным** — Скрытая вкладка не может захватить микрофон, поэтому настоящий микрофон захватывается в контексте интерфейса Steam и передаётся в Discord по локальному WebRTC-соединению.

Вход и выход автоматически следуют за вашим устройством по умолчанию.

## Установка

> **Пока нет в Decky Store.** Ручная установка через режим разработчика.

1. Включите **режим разработчика** в Decky → Общие настройки
2. Перейдите в **Разработчик** в настройках Decky
3. Установите по URL:
   `https://github.com/Necrosiak/Streamcord/releases/latest/download/Streamcord.zip`

### Требование (демонстрация экрана)
Сервер демонстрации использует системный Python + GStreamer. Установите зависимости Python один раз:
```bash
python -m pip install --user aiohttp aiohttp_cors
```

## Сборка из исходников

```bash
git clone https://github.com/Necrosiak/Streamcord
cd Streamcord
pnpm install
pnpm run build
# скопируйте dist/, main.py, defaults/, plugin.json, package.json в ~/homebrew/plugins/Streamcord/
sudo systemctl restart plugin_loader
```
