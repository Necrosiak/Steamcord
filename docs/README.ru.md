# Steamcord

**Discord в игровом режиме Steam** — плагин [Decky Loader](https://github.com/SteamDeckHomebrew/decky-loader) для Steam Deck / Bazzite / SteamOS.

## Как это работает

Steamcord запускает **[Vesktop](https://github.com/Vencord/Vesktop)** — настоящий нативный клиент Discord — невидимо в фоне и управляет им через Chrome DevTools Protocol. Плагин внедряет в него небольшой клиент и выводит всё в **меню быстрого доступа** Steam.

Переход на нативность решает сложные проблемы старого подхода со скрытым браузером: **ваш микрофон и голосовой звук работают нативно**, точно как в десктопном приложении Discord — без трюков с захватом, без обходов автовоспроизведения. Vesktop запускается (и устанавливается при отсутствии) автоматически, остаётся в системе после перезагрузки и никогда не требует окна рабочего стола в игровом режиме.

## Возможности

- **Один Discord на аккаунт Steam (мультисессии)** — У каждого пользователя Steam на машине **свой профиль Discord**: смените аккаунт Steam — и Steamcord автоматически переключит Discord за несколько секунд (в первый раз покажет вход по QR; дальше каждая сессия запоминается). Никто не попадёт в чужой Discord.
- **Вход по QR-коду** — Отсканируйте QR-код приложением Discord на телефоне, чтобы войти мгновенно. На телефоне: *Discord → Настройки → Сканировать QR-код*, затем наведите на код в панели. Без ввода пароля на Deck.
- **Полноэкранный вход (запасной)** — Открывает Discord на весь экран для входа по почте/паролю или решения CAPTCHA, когда QR недоступен.
- **Единая навигация** — Вкладки **Голос / Текст / ⚙️ Настройки** сверху и общий переключатель **Серверы / ЛС** под ними: один и тот же выбор источника действует и для голоса, и для текста.
- **Голосовой чат** — Заходите в голосовые каналы и слышьте всех; каждый участник показан в реальном времени (кольцо речи, значки заглушения/оглушения), ползунок громкости на каждого (0–200 %) **и локальное заглушение на каждого** (заглушите кого-то только для себя, и он об этом не узнает). Микрофон и звук нативные (Vesktop).
- **Личные сообщения (ЛС и групповые)** — Просматривайте беседы и начинайте/присоединяйтесь к голосовым звонкам с друзьями прямо из меню быстрого доступа. Активные звонки выделяются.
- **Голосовой обозреватель серверов** — Смотрите, в каких голосовых каналах есть люди (с аватарами), прежде чем зайти.
- **Текстовый чат — серверы *и* ЛС** — Читайте и отвечайте в канале сервера **или в личной беседе** из QAM (поле ввода на всю ширину, клавиатура Steam открывается сама). **Вложенные изображения показываются миниатюрами** (загружаются только пока канал открыт), а **ссылки открываются в браузере игрового режима**. Автопрокрутка к последнему сообщению.
- **Статус Discord на вашем имени** — Ваше **кликабельное имя пользователя** сверху показывает текущий статус; нажмите на него, чтобы сменить. Необязательная автосинхронизация заставляет Discord **следовать вашему статусу Steam** в фоне; выбор статуса вручную возвращает в ручной режим.
- **Выбор аудиоустройств** — В Настройках выберите устройство **вывода (звук Discord)** и **ввода (микрофон)** — *Авто (системное по умолчанию)* или конкретное, например звук Discord только в **гарнитуру**, пока игра остаётся на HDMI.
- **Заглушить / Оглушить / Отключиться** — Голосовое управление одним касанием из QAM.
- **Демонстрация экрана** — Делитесь всем экраном в голосовом канале (Go Live). Нативно работает в режиме рабочего стола / Big Picture. **В игровом режиме (gamescope) это _бета_:** у gamescope нет портала захвата экрана (обычный Go Live даёт чёрный экран), поэтому отдельная кнопка **«Демонстрация экрана (игровой режим)»** захватывает игру через виртуальную камеру (v4l2loopback), питаемую напрямую из PipeWire-вывода gamescope — единственный работающий там путь захвата. Требуется однократная настройка v4l2loopback.
- **Передача звука игры** — Транслируйте звук вашей игры в голосовой канал **вместе с вашим голосом**. Два ползунка микса (🎙️ голос / 🎮 игра) определяют, что слышат остальные, а вы продолжаете слышать игру как обычно — и это работает даже **без физического микрофона** (плагин создаёт виртуальный вход *Steamcord Mic*).
- **Уведомления в игре** — Входящие звонки в ЛС и упоминания появляются как **нативные уведомления Steam (попап + звук)**, учитывая ваш статус Discord (отключены при невидимом / не беспокоить).
- **Push-to-talk** — С физической кнопкой (по умолчанию R5).
- **Отправка скриншотов** — Отправьте скриншот Steam прямо в открытую беседу.
- **[Vencord](https://vencord.dev/)** встроен в Vesktop, давая доступ к его экосистеме плагинов.

## Установка

> **Пока нет в Decky Store.** Ручная установка через режим разработчика.

1. Включите **режим разработчика** в Decky → Общие настройки
2. Перейдите в **Разработчик** в настройках Decky
3. Установите по URL:
   `https://github.com/Necrosiak/Steamcord/releases/latest/download/Steamcord.zip`

Vesktop устанавливается и запускается плагином автоматически при первом запуске. Достаточно войти один раз (QR или полный экран) — и вы останетесь в системе.

### Требование (демонстрация экрана)
Демонстрация экрана работает сразу — плагин автоматически устанавливает зависимость Python (aiohttp) при первом запуске. GStreamer берётся из системы.

## Сборка из исходников

```bash
git clone https://github.com/Necrosiak/Steamcord
cd Steamcord
pnpm install
pnpm run build
# скопируйте dist/, main.py, defaults/, plugin.json, package.json в ~/homebrew/plugins/Steamcord/
sudo systemctl restart plugin_loader
```
